# Power BI Release Pipeline

Pipeline corporativo para publicação de Power BI com:
- Submissão via GitHub Issues (área de negócio)
- Validações automáticas (BPA / MCP)
- Deploy automático em HML
- Deploy em PROD com dupla aprovação (Sustentação + Owner)
- Auditoria completa via GitHub

Este repositório é a fonte oficial do processo de publicação.
